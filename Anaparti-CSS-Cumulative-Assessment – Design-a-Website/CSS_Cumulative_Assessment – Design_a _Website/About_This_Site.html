About This Site

What This Site Is For

Being able to write down what they are feeling is sometimes the only thing that allows for individuals to express themselves. People often call these collections of writings Diaries. In this day and age, everybody uses technology so, why not have your diary in it too? This site will allow for the user to login and then the user can input their Diary Post which will then be encrypted in the servers so as to not be visible to others. The posts will only be visible by the user and as it is online, it can be accessible by the user globally with out the burden of an actual book.
